# Projet de Fin de Module : Deep Learning (EMSI 2025-2026)

**Étudiant :** RAIS Yasser

## Description du projet
Ce projet intègre l'implémentation, l'expérimentation et l'analyse critique de trois architectures fondamentales en Deep Learning :
1. **MLP (Perceptron Multicouche) :** Classification sur données tabulaires (Breast Cancer).
2. **CNN (Réseau Convolutionnel) :** Classification d'images (Fashion-MNIST) avec étude architecturale.
3. **Seq2Seq (RNN/GRU) :** Traduction automatique (Corpus Tatoeba) avec mécanismes de décodage avancés.

## Structure du Repository
- `/notebooks` : Scripts Jupyter complets utilisés pour l'exécution et les résultats.
- `/src` : Code source modulaire.
- `/docs` : Rapport final en format PDF.
- `/artifacts` : Visualisations (cartes de caractéristiques, courbes de perte) et modèles entraînés.

## Instructions
Tous les notebooks sont configurés pour être exécutables sur Google Colab avec une gestion automatique du device (CPU/GPU).
